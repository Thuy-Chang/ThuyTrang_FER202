# Laptop Store Management

## Custom React Web Application – FER202

Laptop Store Management is a React web application for browsing and managing laptop products. The application supports product CRUD operations, routing, API integration, Redux global state, lazy loading, responsive design and reusable React components.

---

## 1. Application Concept

The application simulates a laptop product management system.

Users can:

- View a list of laptop products
- Search products by name or description
- Add a new product
- View product details
- Edit product information
- Delete a product
- Add products to favorites
- Remove products from favorites
- View the number and total value of favorite products
- Load external laptop recommendations from a public API
