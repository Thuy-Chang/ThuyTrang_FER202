# Lab 6 - Redux Quiz Application

A basic ReactJS quiz application built according to the Lab 6 requirements using:

- Redux
- React Redux
- Redux Thunk
- Redux Toolkit
- Vite

## Main functions

1. Load sample quiz questions through a Redux Thunk action.
2. Add a new question with four answer options and one correct answer.
3. Display questions and options using `map()`.
4. Save the selected option in the Redux store.
5. Highlight the selected option.
6. Check and display correct/incorrect results.

## Run the project

```bash
npm install
npm run dev
```

You can also use:

```bash
npm start
```

Then open the local URL shown in the terminal, normally `http://localhost:5173`.

## Build the project

```bash
npm run build
```
